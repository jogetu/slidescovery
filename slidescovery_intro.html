Slidescovery

まだ見ぬ名作と出会う、発見型スライドショー Discover masterpieces you haven't seen yet, a discovery-style slideshow.

Slidescovery とは？ What is Slidescovery?

Slidescovery は、膨大な画像を スライドショー で軽快に閲覧しながら、
お気に入り仕分け・PNG メタ検索 がサクサク行えるデスクトップアプリです。 Slidescovery is a desktop application that allows you to browse a large number of images in a slideshow,
and easily sort your favorites and search for PNG metadata.

主な機能 Main Features

🖼️ 直感的なスライドショーIntuitive Slideshow

  • 矢印キー / ホイールでパパっと移動Move quickly with arrow keys / wheel
  • 一時停止・再開は p でワンタップPause/resume with a single tap of p
  • スライド間隔を 0.5〜60 秒で自由設定Freely set the slide interval from 0.5 to 60 seconds

🔍 PNG メタデータ検索PNG Metadata Search

  • AI 生成画像の prompt などをリアルタイム表示Real-time display of prompt for AI-generated images
  • 複数キーワード AND 検索 & 自動スキップMulti-keyword AND search & auto-skip

📁 スマートフォルダナビSmart Folder Navigation

  • ツリービュー折りたたみで画面を広々活用Utilize the screen widely by collapsing the tree view
  • ドラッグ＆ドロップで即対象フォルダを登録Register the target folder immediately by drag & drop

✨ 画像仕分けワンクリックOne-click Image Sorting

  • ♥ お気に入り / ★ いいね / 🗑 ゴミ箱 の 3 ルートへ即移動Move to 3 routes immediately: ♥ Favorites / ★ Likes / 🗑 Trash
  • 誤削除防止の確認ダイアログ (ON/OFF 可)Confirmation dialog to prevent accidental deletion (ON/OFF available)

使い方 (クイックスタート) How to Use (Quick Start)

  1. 初回起動で 画像ソースフォルダ を選択Select the image source folder on first startup
  2. スライドショー開始 → 矢印キー / ホイール で移動Start slideshow → Move with arrow keys / wheel
  3. 右上 ℹ️ で PNG 情報パネルを ON/OFFTurn the PNG info panel ON/OFF with ℹ️ in the upper right
  4. 検索バーへキーワード入力でハイライト表示 → 該当画像を高速ジャンプ（⚙️ メニューで「Skip to matching image」）をオンにしてください。デフォルトではオフになっています。）Enter keywords in the search bar to highlight them → Jump to the corresponding image at high speed (Please turn on "Skip to matching image" in the ⚙️ menu. It is off by default.)
